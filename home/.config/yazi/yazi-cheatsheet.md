# Yazi Hotkey Cheat Sheet

## Basic Navigation

| Action | Keys |
|---|---|
| Move up | `h` or `Up` |
| Move down | `j` or `Down` |
| Enter folder / open file | `l` or `Enter` |
| Go to parent folder | `k` or `Backspace` |
| Quit Yazi | `q` |

## File Visibility

| Action | Keys |
|---|---|
| Toggle hidden files | `.` |

## Selection

| Action | Keys |
|---|---|
| Select / unselect file | `Space` |
| Select all | `Ctrl-a` |
| Unselect all | `Ctrl-r` |

## File Operations

| Action | Keys |
|---|---|
| Copy selected files | `y` |
| Cut selected files | `x` |
| Paste files | `p` |
| Delete selected files | `d` |
| Rename file | `r` |
| Create file/folder | `a` |

## Open / Edit

| Action | Keys |
|---|---|
| Open selected file normally | `l` or `Enter` |

## Workflow

Yazi is installed as an on-demand file manager, not as part of the default
herdr workflow. Use `y` to open it directly, or `yy` to open it and change the
shell directory to the selected directory on exit. For editing files, prefer
Neovim file navigation: `Space sf`, `Space gf`, `Space sg`, `Space e`, or `-`.

## Search / Jump

| Action | Keys |
|---|---|
| Search in current directory | `/` |
| Find next match | `n` |
| Find previous match | `N` |

## Help

| Action | Keys |
|---|---|
| Open help | `~` |

## Config Location

| Purpose | Path |
|---|---|
| Dotfiles config | `~/dotfiles/home/.config/yazi/` |
| Real linked config | `~/.config/yazi` |
