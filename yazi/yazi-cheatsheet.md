# Yazi Hotkey Cheat Sheet

## Basic Navigation

| Action | Keys |
|---|---|
| Move down | `j` or `Down` |
| Move up | `k` or `Up` |
| Enter folder / open file | `l` or `Enter` |
| Go to parent folder | `h` or `Backspace` |
| Quit Yazi | `q` |

## File Visibility

| Action | Keys |
|---|---|
| Toggle hidden files | `.` |

## Selection

| Action | Keys |
|---|---|
| Select / unselect file | `Space` |
| Select all | `Ctrl-a` |
| Unselect all | `Ctrl-r` |

## File Operations

| Action | Keys |
|---|---|
| Copy selected files | `y` |
| Cut selected files | `x` |
| Paste files | `p` |
| Delete selected files | `d` |
| Rename file | `r` |
| Create file/folder | `a` |

## Search / Jump

| Action | Keys |
|---|---|
| Search in current directory | `/` |
| Find next match | `n` |
| Find previous match | `N` |

## Help

| Action | Keys |
|---|---|
| Open help | `~` |

## Config Location

| Purpose | Path |
|---|---|
| Dotfiles config | `~/dotfiles/yazi/` |
| Real linked config | `~/.config/yazi` |
