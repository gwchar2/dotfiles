# Contributing

This is a personal dotfiles repository shared publicly for reference.

You are welcome to read, fork, and adapt it for your own setup. Before running
the installer on your machine, review the README's "Make It Yours" section and
change personal paths, aliases, Git settings, and agent instructions as needed.

Bug reports are useful when they include:

- What happened.
- What you expected.
- Steps to reproduce.
- The platform: WSL Ubuntu, macOS, or Windows/WSL.

Feature requests and pull requests may not fit this repository because it tracks
one personal workflow. Forking is the preferred way to make larger changes for
your own environment.
